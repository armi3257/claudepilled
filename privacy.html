Privacy Policy

ClaudePilled Content Tool  ·  Last updated: March 21, 2026

1. Overview

ClaudePilled Content Tool ("the Tool") is a personal, single-user utility for posting content to the @claudepilled TikTok account. This policy describes how data is handled.

2. Data Collection

No user data is collected. The Tool has no users other than its owner. It does not collect, store, or process any personal information from third parties.

3. Data Storage

The Tool stores only locally generated content (images, captions, post metadata) on the owner's private server. No data is shared with any third party except as required to communicate with the TikTok API.

4. TikTok API

The Tool communicates exclusively with TikTok's official API to publish content to the @claudepilled account. Data transmitted to TikTok is governed by TikTok's Privacy Policy.

5. Cookies

This website does not use cookies or any tracking technologies.

6. Personal Use Only

The Tool is strictly for personal use and is not accessible to or operated on behalf of any other individuals or organizations.
